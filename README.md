# MyTermX

跨平台 SSH/SFTP 终端客户端，支持云同步、AI 助手、自动化运维。

## 功能亮点

- **SSH 连接** — 多会话、分屏终端（最多 4 窗格）
- **SFTP/FTP 文件传输** — 断点续传
- **AI 助手** — 通用/Shell/代码审查/运维 4 种模式
- **自动化运维** — 批量执行、定时任务
- **DevOps 集成** — Docker/K8s 管理
- **远程桌面** — VNC/RDP
- **端口转发 & 隧道**
- **插件系统**
- **团队协作** — 工作区共享
- **命令历史 & 代码片段**
- **多主题 & 多语言** — 中/英/日/韩
- **会话恢复** — SQLite 持久化
- **云同步** — JWT + WebSocket
- **跳板机** (Jump Host)
- **本地终端** (PTY)

## 安装指南

### macOS
1. 下载 `.dmg` 文件
2. 双击打开，将 MyTermX 拖入 Applications 文件夹
3. 首次打开如提示安全警告，前往「系统设置 → 隐私与安全性」点击「仍要打开」

### Windows
1. 下载 `.exe` 安装程序
2. 双击运行，按照安装向导完成安装

### Linux
1. 下载 `.AppImage` 文件
2. `chmod +x MyTermX-*.AppImage`
3. `./MyTermX-*.AppImage`

### Android
1. 下载 `.apk` 文件
2. 允许「未知来源」安装
3. 点击 apk 完成安装
